- Drag with right mouse button to turn camera.
- WASD and QE to move
- Space to jump
- F to toggle fly mode
- ESC to toggle mouse lock
- Left click on models to toggle selection.
- Manipulator is shown only on one model in selection.
- If manipulator is shown you can left-drag on it to move the selected model

Explore the menus at the bottom of the screen.
Edit mode (see menu) can be Drag, Select or Add:

 - Drag:    Drag models with left mouse button
 - Select:  Select models by clicking on them.
            Hold shift for multiple selection
 - Add:     Add new model. Select the model from Palette menu.
            New model will match the polygon you hover

When you hover on model, or if you have models selected, you can apply some
simple modelling operations:

- 1 - Sqrt(3) subdivision
- 2 - Subdivide
- 3 - Catmull-Clark subdivision
- 4 - Triangulate
- 5 - Truncate (known to be somewhat broken)
- 6 - Flatten polygon corner normals
- 7 - Smooth polygon corner normals
- Delete - Delete the model
- Space  - Insert new model, matching the polygon you hover

When you hover on polygon: 

- Page Down - Stop physics on hover model
- Home      - Merge currently selected models

- F1 - Toggle rendering selection silhouette
- F2 - Toggle rendering selection wireframe polygon centroid
- F3 - Toggle rendering selection wireframe edges
- F4 - Toggle rendering selection wireframe hidden lines
- F5 - Toggle rendering selection wireframe polygon corner points
- F6 - Toggle rendering translation manipulator
- F7 - Toggle rendering debug information
- F8 - Toggle rendering shadow map visualizations
